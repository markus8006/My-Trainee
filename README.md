# Sylphiette Assistant (sylph_ai)

Assistente virtual personalizavel. Inclui:

- IA com Gemini
- Voz com pyttsx3

## Instalação

```bash
pip install SylphietteIA

```

Uso


---


```python
from Trainee import Model

NOME = Model.Trainee(
    nome,
    personalidade,
    modeloIA,
    API_KEY,
)
```
