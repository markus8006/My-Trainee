# Sylphiette Assistant (sylph_ai)

Assistente virtual personalizavel. Inclui:

- IA com Gemini
- Voz com pyttsx3

## Instalação

```bash
pip install SylphietteIA

```

Uso


---


```python
from Trainee import Trainee


personalidade = (
    "Coloque a personalidade da sua IA"
)
NOME = Trainee.Trainee(
    nome,
    personalidade,
    modeloIA,
    API_KEY,
)
sylph.run()

```
